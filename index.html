Projects

Monthly PnL

Monthly PnL is a React PWA I'm actively working on to help me gain insight into my spending on a monthly basis. Using Firebase as the backing store, the app allows you to enter recurring and one-time payments which are then summarized in a simple view. It's an intentionally manual process, as opposed to connecting with bank APIs, as I want to be cognizant of how much I spend. For the time being, it can be found here.

Workload

A lot of times at work, people will come up to my desk asking about a new feature, or wondering how something should be configured, or letting me know that a trader is having a weird issue, etc., all within the span of an hour. It can be really easy to get sidetracked when this happens, so I wanted to create a super simple nested to-do list to help keep track of what I'm working on. It's written in pure JavaScript and stores all the information in localStorage. The code makes heavy use of ES2015 classes, and still has some bugs to be worked out. It's a bit complex, and definitely makes me appreciate all that React does, but I like to see how far I can get without any libraries. It's published at workload.app.
